# Sockets
Comunicação entre dois tipos de sockets (UDP e TCP)

Ping --> Pong

Pong:
Cria um socket datagrama e espera uma mensagem enviada pelo programa ping
Mensagem: nome ou endereço do ping | n° porto
Enviar uma mensagem (fornecida pelo usuário)

Ping (servidor): Escreve na sua saída o que ele recebeu do Pong
